# Portfolio Website Template

Responsive portfolio website template with smooth animations, project showcase, and contact form.

## Features

- Responsive design
- Smooth scroll animations
- Project showcase section
- About me section
- Skills display
- Contact form
- Social media links
- Modern UI/UX

## Tech Stack

- HTML5
- CSS3 (Grid & Flexbox)
- JavaScript
- Font Awesome Icons

## Customization

1. Edit `index.html` to add your information
2. Modify `style.css` for custom colors and styling
3. Update `script.js` for additional functionality
4. Replace images in `/images` folder

## Note

This repository contains archived work originally created for learning purposes and uploaded later for documentation and portfolio purposes.
